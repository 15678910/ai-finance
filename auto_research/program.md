# 오토리서치 - 포트폴리오 가중치 진화 시스템

## 미션 (Mission)

**기존 몬테카를로 균형형 포트폴리오를 베이스라인으로 시작하여,
가중치 조합을 점진적으로 진화시켜 Sortino Ratio를 최대화한다.**

## 황금 지표 (Golden Metric)

```
Sortino Ratio = (연수익률 - 무위험금리) / 하락 변동성
```

단일 스칼라 값으로 평가. 다른 모든 지표(Sharpe, MDD, 변동성)는 참고용.

## 절대 규칙 (Absolute Rules)

### 🚨 실전 매매 절대 금지
이 시스템은 **시뮬레이션 전용**이다. 출력 결과는:
- ✅ 가중치 추천만 표시
- ✅ 사용자가 직접 검토
- ❌ 자동 매매 절대 불가
- ❌ 브로커리지 API 연동 절대 불가

### 평가 격리 (Evaluation Isolation)
백테스트 평가 함수는 진화 루프와 완전히 분리되어야 한다.
AI(또는 진화 알고리즘)가 평가 코드를 수정할 수 없도록 격리한다.

### 과적합 방지 (Overfitting Prevention)
- **In-sample (학습)**: 과거 1년 데이터로 가중치 진화
- **Out-of-sample (검증)**: 학습 데이터에 포함되지 않은 6개월로 평가
- 검증 Sortino > 학습 Sortino × 0.7 미만이면 채택 거부

### 시간 예산 (Time Budget)
- 단일 실행: 최대 5분
- 평가당 시도: 최대 50회

### 안전망 (Safeguards)
- 가중치 합 = 1.0 (정규화 강제)
- 단일 종목 최대 비중: 50%
- 단일 종목 최소 비중: 1% (완전 제외 방지)

## 진화 루프 (Evolution Loop)

```
1. 베이스라인 로드 (몬테카를로 균형형 결과)
2. 현재 최적 가중치 W를 ±10% 범위로 변형 → W'
3. W'로 백테스트 실행 → Sortino 계산
4. 검증 데이터로도 검증 → 과적합 체크
5. 개선되었다면 → 채택, 아니면 → 폐기
6. 반복 (시간 예산 소진까지)
```
